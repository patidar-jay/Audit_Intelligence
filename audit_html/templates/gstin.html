🔍 GSTIN Validator

Validate Indian GST Numbers — Format check + State identification

GSTIN FORMAT GUIDE
PositionCharactersDescription
1–2DigitsState Code (01–37)
3–7LettersPAN first 5 characters
8–11DigitsPAN digits
12LetterPAN last character
131–9 or A–ZEntity number
14ZAlways Z
15Digit/LetterChecksum

Example: 27AAPFU0939F1ZV → Maharashtra
